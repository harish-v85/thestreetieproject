## How to log a feeding activity for one dog

1. Open **Dogs** → **All dogs**, then choose a dog.
2. On their profile, scroll to **Feeding log**.
3. Click **Log a feeding**, fill in when you fed them and any notes, then submit.
4. Optionally set **Feeding location (optional)** using **Add map location** or **Add coordinates** (same behaviour as on the full edit form). You can leave this empty.

## How to log a feeding activity for a group of dogs

1. Open **Dogs** → **Feeding Activity**.
2. Under **Select dogs**, tick each dog you fed.
3. Set **When**, **Notes**, and optionally **Feeding location** using **Add map location** or **Add coordinates** (same as on a dog profile). You can leave location empty.
4. Click **Save feeding for selected dogs** so each selected dog gets one entry for that visit.

## How to add a new dog

1. Open **Dogs** → **Manage dogs**.
2. Click **+ Add dog** near the top of the list.
3. Work through the form: profile, location, buddies, photos, medical, and any other sections you need.
4. Save when finished.

## How to edit a dog’s profile

1. Open **Dogs** → **Manage dogs**.
2. Find the dog in the list and open **Edit** (or the equivalent action).
3. Move between the sections — Profile, Location, Buddies, Photos, Medical, and so on — and change what you need.
4. Save the page so the public dog profile stays up to date.

## How to update welfare check status from a dog’s profile page

1. Open **Dogs** → **All dogs**, then click the dog to open their public profile.
2. Scroll to **Current Welfare Status**.
3. Click **Update latest welfare status**. 
4. In the form, open **Welfare check** and choose a status, and add any text under **Welfare check — remarks**. The **Update welfare status** button stays disabled until you enter something in at least one of these fields.
5. Click **Update welfare status** to save.

## How to add a medical record from a dog’s profile page

1. Open **Dogs** → **All dogs**, then click the dog to open their profile.
2. Scroll to **Medical records**.
3. Click **Add a new medical record**.
4. Fill in **Event type**, **Date of event**, optional **Next due**, and **Notes**, then click **Add medical record**.

## How to add a Welfare check for a dog (from Manage dogs)

1. Edit the dog from **Manage dogs** as above.
2. In **Profile**, find **Welfare check** (status) and **Welfare check — remarks**.
3. Update them to match the latest visit, then save. Visitors see this on the dog’s public profile.

## How to update Medical records for a dog (from Manage dogs)

1. Edit the dog from **Manage dogs**.
2. Scroll to the **Medical** section.
3. Add a new record or adjust existing ones (super-admin only for edit/delete of existing rows), then save. Those details show on the dog’s public profile.

## How to add a dog’s age (estimate)

1. Open **Dogs** → **Manage dogs**, then **Edit** for the dog.
2. In **Profile**, find the **Age (estimate)** box.
3. Set **Estimated birth year**, **Age estimated on** (when that estimate was recorded), and **Age confidence** (vet-assessed, best guess, or unknown).
4. Save the page. The app derives the dog’s age from the birth year over time; it is not stored as a single fixed “age in years” number.

## How to mark the hangout location using the map

1. Edit the dog from **Manage dogs**.
2. Open the **Location** section.
3. Under **Hangout map location**, click **Add map location** if no pin is shown yet (if a map already appears, you can click the map to move the pin).
4. Click the map to place or move the pin, drag the pin, or use your current location if your browser offers it.
5. Save the page. The pin is what visitors see on the dog’s public profile map.

## How to mark the hangout location using coordinates

1. Edit the dog from **Manage dogs**.
2. Open the **Location** section.
3. Under **Hangout map location**, click **Add coordinates**.
4. Enter **Latitude** and **Longitude**, then click **Use coordinates** to place the pin on the map. (You can find the lat-long values from Google Maps. If you're using the Google Maps site on a desktop/laptop, right click the desired point. If you're using the Google Maps app on your phone, long press the desired point.)
5. Save the page.

## How to add a new Locality

1. Open **Locations** → **Localities** in the top navigation.
2. Click **+ Add locality** to create one, or open a row to edit an existing locality.
3. The new locality is sent to the super-admin for approval. Once approved, it becomes public.

## How to add a new Neighbourhood

1. Open **Locations** → **Neighbourhoods** in the top navigation.
2. Click **+ Add neighbourhood** to create one, or open a row to edit an existing neighbourhood.
3. The new neighbourhood is sent to the super-admin for approval. Once approved, it becomes public.
